## Configuração

Para configurar cada integração, adicione as variáveis de ambiente correspondentes:

```env
# Plaid
PLAID_CLIENT_ID=seu_client_id
PLAID_SECRET=seu_secret
PLAID_ENV=sandbox|development|production

# GoCardless
GOCARDLESS_SECRET_ID=seu_secret_id
GOCARDLESS_SECRET_KEY=sua_secret_key

# Slack
SLACK_CLIENT_ID=seu_client_id
SLACK_CLIENT_SECRET=seu_client_secret

# OpenAI
OPENAI_API_KEY=sk-...

# Mistral
MISTRAL_API_KEY=...
```

## Webhooks

O Mid Poker pode receber webhooks de serviços externos para processamento em tempo real:

- **Postmark**: Recebimento de emails no inbox
- **Plaid**: Atualizações de transações
- **Stripe/Polar**: Eventos de pagamento (se configurado)

Consulte a [documentação da API](/api-reference/engine/introduction) para mais detalhes sobre os endpoints de webhook.
