# MAPEAMENTO COMPLETO - ABA GERAL (A até AV)

## Identificação do Jogador (A-I)

| Coluna | Campo |
|--------|-------|
| A | (vazio/índice) |
| B | ID do jogador |
| C | País/região |
| D | Apelido |
| E | Nome de memorando |
| F | Agente |
| G | ID do agente |
| H | Superagente |
| I | ID do superagente |

## Classificações (J-N)

| Coluna | Campo |
|--------|-------|
| J | Ganhos de jogador gerais + Eventos |
| K | Classificação PPSR |
| L | Classificação Ring Game |
| M | Classificação de RG Personalizado |
| N | Classificação MTT |

## Ganhos do Jogador (O-X)

| Coluna | Campo |
|--------|-------|
| O | Geral |
| P | Ring Games |
| Q | MTT, SitNGo |
| R | SPINUP |
| S | Caribbean+ Poker |
| T | COLOR GAME |
| U | CRASH |
| V | LUCKY DRAW |
| W | Jackpot |
| X | Dividir EV |

## Tickets (Y-AA)

| Coluna | Campo |
|--------|-------|
| Y | Valor do ticket ganho |
| Z | Buy-in de ticket |
| AA | Valor do prêmio personalizado |

## Taxas (AB-AG)

| Coluna | Campo |
|--------|-------|
| AB | Geral |
| AC | Taxa |
| AD | Taxa (jogos PPST) |
| AE | Taxa (jogos não PPST) |
| AF | Taxa (jogos PPSR) |
| AG | Taxa (jogos não PPSR) |

## SPINUP & Caribbean (AH-AK)

| Coluna | Campo |
|--------|-------|
| AH | Buy-in de SPINUP |
| AI | Premiação de SPINUP |
| AJ | Apostas de Caribbean+ Poker |
| AK | Premiação de Caribbean+ Poker |

## Ganhos do Clube (AL-AQ)

| Coluna | Campo |
|--------|-------|
| AL | Apostas do COLOR GAME |
| AM | Premiação do COLOR GAME |
| AN | Apostas (CRASH) |
| AO | Prêmios (CRASH) |
| AP | Apostas de LUCKY DRAW |
| AQ | Premiação de LUCKY DRAW |

## Jackpot e Finais (AR-AV)

| Coluna | Campo |
|--------|-------|
| AR | Taxa do Jackpot |
| AS | Prêmios Jackpot |
| AT | Dividir EV |
| AU | Valor do ticket entregue |
| AV | Buy-in de ticket |
