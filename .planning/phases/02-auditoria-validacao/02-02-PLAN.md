Analisar como dados são transformados da planilha para entidades do banco.

1. **Mapeamento Planilha → Database:**

| Aba da Planilha | Tabela DB | Transformações |
|-----------------|-----------|----------------|
| Geral | poker_player_summary | ? |

2. **Verificar cálculos matemáticos:**
   - chip_balance = sum(transactions)
   - total_rake = sum(session_players.rake)
   - rakeback_amount = gross * rakeback_percent / 100
   - Qualquer outro cálculo derivado

4. **Verificar conversões de tipo:**
   - String → Number (valores monetários)
   - Date parsing (formatos aceitos)
   - ID normalization (PPPoker IDs)

5. **Identificar transformações implícitas:**
   - Defaults aplicados
   - Valores calculados vs importados
   - Arredondamentos
