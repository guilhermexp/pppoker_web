Bem-vindo à documentação do Mid Poker! Esta plataforma foi desenvolvida para ajudar jogadores de poker a gerenciar suas finanças de forma eficiente e profissional.

## O que é o Mid Poker?

Mid Poker é uma plataforma completa de gestão financeira projetada especificamente para jogadores de poker. Oferece:

- **Rastreamento de Sessões**: Controle em tempo real das suas sessões
- **Gestão de Bankroll**: Acompanhamento completo com métricas e alertas
- **Magic Inbox**: Correspondência automática de comprovantes
- **Assistente IA**: Insights personalizados sobre sua performance

## Self-Hosting

Você pode hospedar o Mid Poker em sua própria infraestrutura para maior controle sobre seus dados. Este guia irá orientá-lo no processo completo de configuração.

## Pré-requisitos

Antes de começar, certifique-se de ter:

- Uma conta [GitHub](https://github.com/)
- Uma conta [Supabase](https://supabase.com/)
- Uma conta [Vercel](https://vercel.com/) (opcional, para deploy)
- [Bun](https://bun.sh/) instalado localmente
